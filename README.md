# QuantumCrypt - Advanced Quantum Encryption Platform

## 🚀 Overview

QuantumCrypt is a cutting-edge, cyberpunk-themed web application for encrypting and decrypting **text**, **images**, and **videos** using advanced quantum-inspired encryption algorithms. Built with a futuristic, glitchy, and neon aesthetic, this platform offers a seamless user experience with smooth animations and responsive design.

## 🎯 How to Use

### Getting Started

1. **Open the Landing Page**
   - Start with `landing.html`
   - Click "LAUNCH ENCRYPTOR"

2. **Experience the animation**
   - Phase 1: Quantum initialization (3 sec)
   - Phase 2: Cyberpunk city (2.5 sec)
   - Phase 3: Matrix reveal (2 sec)

### Encrypting Data

#### Text Encryption
- Select "TEXT", enter your message and quantum key, click "ENCRYPT"

#### Image Encryption
- Select "IMAGE", drag/drop or choose a file, enter quantum key, click "ENCRYPT", download/copy result

#### Video Encryption
- Select "VIDEO", drag/drop or choose a file, enter quantum key, click "ENCRYPT", download result

### Decrypting Data

- Go to "DECRYPT" (enhanced-decrypt.html)
- Select TEXT/IMAGE/VIDEO
- Provide encrypted data and quantum key, click "DECRYPT", view/download result
